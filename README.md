# ನಲ್ಲ-ನುಡಿ (Nalla-Nudi)

A Bridge Dictionary for Technical Terms built for Kannada-medium students.

## Features
- Offline search in under 200ms
- English to Kannada bilingual terms
- TTS Pronunciation
- Subject filters (Science, Math, Commerce)
- My List and Flashcard Revision
- 100% Offline

## Tech Stack
- Kotlin + Jetpack Compose
- Room Database (FTS4)
- Hilt DI
- MVVM Architecture
- Android TTS
